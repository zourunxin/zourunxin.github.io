win10 装机

准备

  • win 电脑重要数据备份（存到磁盘）；
  • win 电脑下载 win10 iso 镜像到 u 盘。

官方方法装机

  1. 将 u 盘设为第一启动项：
    • 快捷键法：华硕笔记本 – 开机后一直按 esc，进入界面后选中自己 u 盘 + UEFI/Legacy 引导模式（无 UEFI 前缀的为 Legacy 引导模式）装机。
    • BIOS 法：开机后一直按 delete 键进入 BIOS 界面（F7 切换简单/复杂模式），把自己的 u 盘 + UEFI/Legacy 引导模式拖到最前面/最上面，按 F10 保存设置并重启。
  2. 启动后简单设置 – 点击现在安装
  3. 激活界面选“我没有产品密钥” – 选 win10 专业版
  4. （自定义）仅安装 windows – 磁盘分区（若有 100G 情结，分区时存储空间填 103056/103088）
  5. 点击 c 盘 – 下一步
  6. 自动安装后读秒重启
  7. 若是前面进入 BIOS 设置 u 盘启动，则重启需重新进入 BIOS，将 c 盘放在最前面/最上面启动系统；若使用快捷键法设置 u 盘启动，则无需其他操作，直接重启即可。
  8. 后面按需要个性化配置（不联网） – 继续执行有限设置 – 个性化配置
  9. 开机完成
  10. 桌面鼠标右击 – 个性化 – 主题 – 桌面图标设置 – 添加‘此电脑’图标
  11. 右击‘此电脑’ – 管理 – 磁盘管理 – 选定盘符 – 右击 – 更改驱动器号和路径 – 添加 – 再右击盘符格式化 – 完成分配盘符
  12. 专业版需激活，家庭版不需要（因为本机是家庭版）。

硬盘因进程占用无法弹出

  1. 进入“事件管理器”（全局搜索），查找最近的事件（指“进程占用”的弹窗）。若没有，则再试一次弹出硬盘，出现该弹窗后到事件管理器按 F5 刷新后可见；
  2. 找到该事件提示的占用进程及 pid；
  3. 到任务管理器中停止该进程任务。若无法停止该任务，右击该进程进入所在文件夹，确认其所在软件后关闭软件，再停止该任务；
  4. 停止任务后即可弹出硬盘。

adb 禁用 android 程序

参考adb 下载安装及使用

条件准备

  • adb 程序、环境变量配置
  • android 手机、开发者调试模式

常用命令

以 android 应用商店为例

  • 查看 adb 版本：adb version
  • 连接设备：adb devices，需手机端允许
  • 断开设备：adb kill-server
  • 查看所有应用列表：adb shell pm list packages
    • 查看系统应用列表：·adb shell pm list packages -s
    • 查看第三方应用列表：adb shell pm list packages -3
  • adb 停用指定应用：adb shell pm disable-user com.bbk.appstore（优先考虑）
  • adb 启用指定应用：adb shell pm enable com.bbk.appstore
  • adb 卸载指定应用：adb shell pm uninstall -k --user 0 com.bbk.appstore
    • -k：卸载应用且保留数据与缓存
    • -–user：指定用户 id，Android 系统支持多个用户，默认用户只有一个，id=0（一般不使用，否则机器上还有这个应用，只是该用户用不了了）
  • adb 恢复被卸载应用：adb shell pm install-existing --user 0 com.bbk.appstore
  • adb 安装指定应用：adb install .apk（安装电脑下载的 apk，拖到 cmd 中 ）

报错

安装指定 apk 报 200 错误：adb: failed to install toutiao.apk: Failure [-200]
原因：安装时需在手机中及时输入密码允许安装

安装指定 apk 报 error: INSTALL_FAILED_VERSION_DOWNGRADE
原因：设备中已安装同名 app，且本次的版本并不高

老爸手机做的更改

  • app store(com.bbk.appstore) 被卸载，可通过命令行重新安装
  • global search(com.vivo.globalsearch) 被卸载，不可通过命令行重新安装
  • 浏览器(com.vivo.browser) 被卸载
