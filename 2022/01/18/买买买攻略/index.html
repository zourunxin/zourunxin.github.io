买买买攻略

眼镜

镜片

品牌： 蔡司 > 依视路 > 豪雅（进口）；明月（国产）
蔡司系列：佳锐 < A系列莲花膜 <（前面的不推荐）（后面的都含铂金膜） 新清锐 < 臻锐 < 铭睿
依视路系列：白晶 < 钻晶A+ <（前面的已被淘汰） 钻晶A3（性能最好，但是不防蓝光） < 钻晶A4（含蓝光） < X4 < X6（依视路要小心可能是被收购的品牌镜片）

基材：

制作镜片的材料，它影响镜片的透光率、阿贝数、折射率

折射率：1.56 < 1.61 < 1.67 < 1.74，折射率越高镜片越薄价格也就更贵，不过度数不高没必要配薄镜片，因为折射率越高清晰度（阿贝数）越差。
阿贝数：阿贝数指的是色散系数，它是衡量镜片成像清晰度的重要指标。其数值越高，镜片的成像越清晰。其数值越低，镜片的成像越模糊。它还和镜片自身的折射率也是息息相关的。一般来说，镜片的折射率越高，阿贝数就越低。
耐磨性：树脂材质
基本条件：防紫外线、防辐射
防蓝光适用我

镀膜工艺：好的镜片膜层耐油污、耐磨损、耐腐蚀，防静电抗反射，镜片就能高清、易清洁、防尘。

镜框

品牌
高端品牌（3k+）：奥利地诗乐、IC!BERLIN、日本夏蒙
中端品牌：派丽蒙(PARIM)、宝姿、暴龙、保圣、陌森、施洛华、索菲亚、海伦凯勒、LOHO

材料
板材：强度高经久耐用，重量相对较重
TR90：超轻、超韧性、耐撞击
钛合金：韧性好，性价比高的材质，不过时间久了可能会掉漆。
纯钛：重量超轻，航空级材料，IP电镀工艺不会掉漆与人体皮肤接触不会过敏，耐酸碱、耐腐蚀，价格最贵。

羽毛球拍

羽毛球拍图解

我的选择：重量 4U，平衡点头重型，中杆硬，磅数 25 磅，方型 ISO 拍头，耐打型的线。

重量

在羽毛球拍里，U 表示重量，U=-5 克，100 克是基准重量，3U = 100-15 克，也就是该羽毛球拍的重量在[85,90) 区间内。

关于 U 的选择：

  • 成年男生一般都选择4U；
  • 力量大，喜欢进攻，手腕用力娴熟的人一般选择3U；
  • 力量小，喜欢技术控球的一般选择5U;
  • 女生选择5U以上的占多数；

关于平衡点

平衡点：是指把球拍放在手指头上左右移动，两端保持平衡的那一点就是平衡点。平衡点高，头重；平衡点低，头轻（高指的是平衡点靠近拍框）。两个重量相同的拍子，头重的拍子挥重大，头轻的挥重小。

类型 平衡点范围
头重型 大于 295mm
平衡型 285 - 295mm
头轻型 小于285mm

适应人群：

  • 头重型适合追求羽毛球拍进攻性强的球友
  • 平衡型适合追求羽毛球拍各方面性能均衡的球友
  • 头轻型适合追求挥拍速度快的球友

中杆

中杆软硬是影响羽毛球拍手感的主要原因之一

拍杆软 拍杆硬
弹性好 弹性差
拉高远球可以借力 难借力，用多少力，传递多少力
处理被动球有优势，平抽挡有优势，适合防守 主动进攻有优势，适合进攻
控制落点难，落点不精准 落点精准，指向明确
对发力要求低 对发力要求高
适合初学者及力量小的选手 适合力量较大，善于发力的球员（老手）

磅数

磅数是指：羽毛球拍穿线后，线对拍框的拉力，用 lbs（磅）表示；

指标 磅数
低磅 ≤20磅
低中磅 20磅 - 23磅
中磅 23磅 - 25磅
中高磅 25磅 - 27磅
高磅 ≥27磅
  • 低磅（≤ 20磅）
    优点：无论用什么线，羽毛球拍弹性都很好，缺点：粘球，球的方向控制不好
  • 低中磅（20磅 - 23磅）
    优点：弹性好，控球不错，缺点：击球滞留感、进攻威力低
  • 中磅（23磅 - 25磅）
    优点：羽毛球拍和拍线都能发挥其特有性能，缺点：比较均衡没有哪方面特别突出
  • 中高磅（25磅 - 27磅）
    优点：熟练后用多少力就有多少速度，小球也能控制，缺点：新手拉后场吃力，击球无力
  • 高磅（≥ 27磅）
    优点：用多少力就有多少效果，击球精准，缺点：手腕控制力和爆发力要够，不然不容易控制球路，容易伤手腕

羽毛球拍磅数怎么选择

  • 新手一般拉23-25磅
  • 力量小的女生拉20-23磅
  • 腕力可以灵活运用，且是进攻型风格的，选择25-27磅
  • 进攻型风格的高手选择27磅以上

甜区

甜区：甜区是指羽毛球拍面最适合击球的区域，在这个区域击球时击球效果好，可以符合击球人的意愿；

拍线

羽毛球线主要要考虑弹性、耐打两种因素

  • 打球频率高，喜欢暴力击球的球友建议选择耐打型的线
  • 力量较小或者女生选择弹性好的线
  • 喜欢双打以及均衡型选手建议选择耐打性和弹性较均衡的拍线
  • 喜欢暴力型击球的可以选择较硬的拍线

乳胶漆

乳胶漆标准工艺：一遍底漆 + 两遍面漆（1 升可以涂刷一遍 10~12 平米的墙面）
墙漆施工面积 =（建筑面积 × 80% - 10）× 3。建筑面积是指购房面积，现在的实际利用率一般在80%左右，厨房、卫生间一般是采用瓷砖、铝扣板的，该部分面积大多在10平方米。这样计算出的面积最接近实际使用面积。乘 3 后即为需刷墙面面积。

底漆和面漆的区别

  1. 成分不同
    • 底漆的主要成分：树脂、溶剂、助剂、填料。
    • 面漆的主要成分：树脂、溶剂、助剂，没有填料。
  2. 作用不同
    • 底漆的主要作用：
      1. 底漆是第一道工序，有些墙面湿气重或则碱性大，底漆可以有效隔离开来。
      2. 填平：有些墙面不平整，使用后可以减少面漆的使用，也使面漆更容易施工。
      3. 附着力：底漆中的乳液含量高，可以提高乳胶漆膜与墙面的结合力。
    • 面漆的主要作用：
      是最后涂抹的一层，主要起装饰和保护作用，面漆的质量直接影响整个漆膜的质量，所以在选购时通常面漆搭配底漆，买之前要先咨询好卖家。

乳胶漆施工步骤

批腻子——打磨——修补——刷底漆——涮面漆——涮面漆

  • 腻子粉一般要刷3遍，完成后再仔细检查一遍，如果不平整用砂纸打磨好。
  • 在刷乳胶漆前最后一道工序是清理墙面的灰尘颗粒，可以用鸡毛毯子、吸尘器。假如灰尘没清理干净而去刷漆，容易造成起皮、掉落等情况。
  • 底漆一遍 + 面漆两遍，每次等干透好再刷第二遍。乳胶漆兑水比例严格遵照说明书。

如何挑选乳胶漆

  1. 环保性能

    最基础的是国标：VOC含量≤80g/L
    中国十环认证：VOC含量≤50g/L
    以下是世界四大严苛环保标准
    美国绿色卫士（金级认证）
    欧洲 IAC（金级认证）
    德国 GEV
    法国室内A+认证：VOC含量≤2g/L
    VOC：会产生危害的那一类挥发性有机物，比如甲醇、氨、乙二醇等

    根据国家标准 GB 18582-2020《室内装饰装修材料内墙涂料中有害物质限量》的要求，室内涂料的环保性能主要有两个指标：

    • 游离甲醛含量<50mg/kg
    • 水性内墙VOC（可挥发性有机物）含量<80g/L

  2. 漆面性能

    • 耐擦洗性：墙面污渍和小孩乱画都能有效的去除，污垢无法紧附，延长使用寿命长，可擦洗次数越高，性能越好。
    • 遮盖性：指的是涂料中的颜料能遮盖被涂物体表面不再透过涂膜而显露的能力，遮盖性越高，工作时间越少，用量也更省，装饰效果越好。

  3. 功能性：

    • 基础功能有防霉、抗碱、净味。
    • 进阶功能有抗甲醛、抗菌、抗污。
    • 五合一：防霉、抗碱、抗菌、耐刷洗、环保
    • 功能越多价格越高，产品通常命名3合1、5合1、全效。
    • 功能越多意味着添加的助剂也就越多。

  4. 乳胶漆使用注意

    • 为避免乳胶漆浪费，在刷之前先算好使用面积。
    • 同品牌的乳胶漆要用同品牌的专用色调。
    • 调颜色要先从浅到深，小面积试色，与色卡颜色一致再进行大面积涂刷。
    • 刷的时候后不能开窗通风，要保持阴干 3 到 4 天
    • 刷完后留一点底漆，以防以后补漆使用。
    • 刷完漆后最好间隔个半个月再入住。

乳胶漆品牌

  • 立邦是亚洲规模最大的涂料企业，品类齐全，影响力大
  • 多乐士是荷兰 ICI 集团旗下的高端品牌，是全球最大的油漆生产商之一。
    多乐士和立邦是国内乳胶漆的两大霸主。

电动牙刷

关注的点：刷头、续航时间、挡位调节、静音、防水。

电机

震动频率：建议选购 31800-40000次/min 的（新手应选偏低的）
电动牙刷电机主要分为普通电机和磁悬浮电机。

  • 普通电机： 动力强悍，但震动很没有规律，噪音大，出现在廉价电动牙刷里面。这种电机对牙齿伤害很大，轻则不适应，重则牙龈出血等。
  • 磁悬浮电机：噪音小，寿命高

刷毛

  • 磨圆率：刷毛顶部的磨圆情况。成人不得低于 70%，儿童不得低于 90%
  • 刷毛种类：主流且目前最好的是杜邦刷毛，此外，还有韩国 3mm 高低差刷毛
